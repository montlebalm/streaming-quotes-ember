# Streaming Stocks with Ember

This is an example of how you can simulate real-time stock updates using Ember.

## Usage

Run `ember serve` from the project root.

## Setup

### Requirements

- [ember-cli](http://www.ember-cli.com/)
- [npm](https://www.npmjs.org/)
- [bower](http://bower.io/)

### Installation

- `bower install`
- `npm install`
